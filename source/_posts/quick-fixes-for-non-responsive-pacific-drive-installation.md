### _Firefox is already running, but is not responding_

 If you‚Äôre now experiencing this problem, then you‚Äôve probably seen an error message like this:

![](https://images.drivereasy.com/wp-content/uploads/2019/11/image-13.png)

 The most straightforward way is to click**Close Firefox** , but sometimes it‚Äôll give no response and Firefox just won‚Äôt shut down. In this case, you should:

#### Fix 1: Close Firefox via Task Manager

 1) On your keyboard, press the**Windows logo key** and**R** at the same time to fire up the Run dialog box. Type**taskmgr** and click**OK** .

![](https://images.drivereasy.com/wp-content/uploads/2019/11/image-14.png)

 2) On the**Processes** tab, scroll through the list until you‚Äôv found**Firefox** . Select to highlight it, then click**End task** .

![](https://images.drivereasy.com/wp-content/uploads/2019/11/image-15.png)

 Exercise caution with this step in case you mistakenly close some important system processes.

3) After the Firefox has been completely closed, relaunch it.

 Wait to see if Firefox continues to present you with the same error message. If it does, try out Fix 2.

#### Fix 2: Check your access permission to the profile folder

 1) Launch Firefox and click**Open menu** at the upper right corner.

![](https://images.drivereasy.com/wp-content/uploads/2019/11/image-16.png)

![](https://images.drivereasy.com/wp-content/uploads/2019/11/image-17.png)

![](https://images.drivereasy.com/wp-content/uploads/2019/11/image-18.png)
